The one action you need to take after deploying:
Go to search.google.com/search-console, add your property ctrlbay.in, then go to Sitemaps → Enter sitemap URL → https://ctrlbay.in/sitemap.xml and hit Submit. That's it — Google will start crawling within 24–48 hours and your pages will begin appearing in search results.
Without that submission step, the sitemap file exists but Google doesn't actively know to look for it.
